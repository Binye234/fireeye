## 项目介绍
**火眼敏感词监测系统**是专门对政府类网站实施网站内容敏感词监测的工具，对网站内容主要以  
html静态为主要内容的效果最佳，可以有效匹配出敏感词内容。  
本项目主要由JAVA语言开发，开发框架为springboot，数据库为mysql。
